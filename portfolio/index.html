Susan Potter's Public Portfolio

Open Source Software (OSS) Projects

Susan Potter primarily works on proprietary business services and/or applications and as such most of her work is not available for public review. However, she has been involved in a few open source projects from founding/lead developer on a project to bug fix patch contributor.

The following is a sampling of her open source endeavors to give potential clients an opportunity to evaluate her hands on development skills.

Twitter4R
Ruby, Twitter, REST, API, Rubyforge, Twitter4R, OSS
Lead and founding developer and manager for the open source Ruby library for the public Twitter REST API. Current release is 0.6.0, with 0.6.1 scheduled for release in Feb 2011. Project active since early 2007.
Rebar Templates
Erlang, Utilities, Rebar, Templates
Rebar templates for Erlang and OTP file types. Includes my own templates for gen_server, gen_event, escript, etc.
MOS System Utilities
Erlang, Utilities
Managed Open Source (MOS) system utilities. Anything from generating 'random' passphrases, generating code from templates, etc.
Metafusion Crypto
Ruby, Cryptography, Security, Digital Signature, Private Key, Paypal Encryption
An open source library written in Ruby that makes private key and digital signature encryption much simpler and less error prone.
