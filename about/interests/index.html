About Susan Potter

Professional Interests

Susan has a number of professional interests including:

  • Distributed, decentralized and fault-tolerant systems with the goal to improve availability and robustness of services and applications.
  • Designing and implementing Domain Specific Languages (DSL) for business applications to improve readability and testability of software.
  • Automating deployments of complex distributed services from static prescriptions to dynamic provisioning, configuration and integration of new nodes based on demand.
  • Improving software development processes on smaller teams (less than ten members) in practice. Consistently managing software release cycles on different levels; from features to codebase to quality assurance to deployment and production support.
  • Pragmatically using software (codebase, runtime and business) metrics Codebase metrics include: Halstead measures, Cyclomatic complexity, code coverage measures, etc. Runtime metrics include: response times, load measurements, uptime/availability, resource metering, etc. Business metrics include: conversions, usage patterns (usability), marketing metrics, planning metrics, etc.
  • Contributing to Open Source Software (OSS) projects with code, tests, insight or documentation.
