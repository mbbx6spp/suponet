Agile Holiday Reading

2006-12-11 09:53:00 -0600
Over the weekend a friend who is just starting to head up a new web product team asked me for for some pointers on good agile project/methods books. I'll try to make my choices as technology agnostic as possible, but some of the material might be more specific to my friend's technology stack (Java/Spring/Hibernate). This is different. %a{:href => "http://www.amazon.com/gp/product/0321146530?ie=UTF8&tag=illinilistn09-20&linkCode=as2&camp=1789&creative=9325&creativeASIN=0321146530"} Test Driven Development: By Example (Addison-Wesley Signature Series) %img{:alt => "", :border => "0", :height => "1", :src => "http://www.assoc-amazon.com/e/ir?t=illinilistn09-20&l=as2&o=1&a=0321146530", :style => "border:none !important; margin:0px !important;", :width => "1"}/ %br/ %small by Kent Beck - Addison-Wesley Professional; 1st edition (November 8, 2002) %div Good first read for those just starting TDD. Not for the TDD guru! %a{:href => "http://www.amazon.com/gp/product/0201485672?ie=UTF8&tag=illinilistn09-20&linkCode=as2&camp=1789&creative=9325&creativeASIN=0201485672"} Refactoring: Improving the Design of Existing Code %img{:alt => "", :border => "0", :height => "1", :src => "http://www.assoc-amazon.com/e/ir?t=illinilistn09-20&l=as2&o=1&a=0201485672", :style => "border:none !important; margin:0px !important;", :width => "1"}/ %br/ %small by Fowler, Beck, Brant, Opdyke, Roberts - Addison-Wesley Professional; 1st edition (June 28, 1999) %div Great book whether the project already has a large existing code base or if you are just starting out, refactoring is almost the most important part of agile software projects. As long as your remember that without the test assets agile software products should possess, refactoring would be (and is for many non-agile developers) highly error prone. Remember the refactoring strategies outlined in this book heavily rely on being test-infected on your project. Don't try this at home without having a solid arsenal of unit, functional and integration tests for your product. %a{:href => "http://www.amazon.com/gp/product/0131482394?ie=UTF8&tag=illinilistn09-20&linkCode=as2&camp=1789&creative=9325&creativeASIN=0131482394"} Agile Java(TM): Crafting Code with Test-Driven Development (Robert C.
