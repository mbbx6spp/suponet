Twitter4R v0.2.0: Timeline API

2007-07-08 21:06:00 -0500
The %b Timeline API segment of Twitter4R provides access to the REST API that deals with public, friend and [public] user status timelines. For example, to get an %tt Array of statuses representing your own timeline on Twitter all you would need to type was: %pre %code :preserve gem('twitter', '>=0.2.0') require('twitter') client = Twitter::Client.new(:login => 'mylogin', :password => 'mypassword') timeline = client.timeline_for(:me) If your timeline is public and you do not need to invoke any %em authenticated methods, you do not need to pass in login/password credentials to the %tt Twitter::Client constructor, just %tt Twitter::Client.new is all that is needed. Other than the %tt :me timeline, the following are available: %ul %li :public - returns the public timeline on Twitter (last 20 statuses) %li :friends - returns a timeline of all your friends' statuses %li :friend - returns a timeline of one particular friend %li :user - returns timeline of one particular user Different options are available with each of the timelines described above.
