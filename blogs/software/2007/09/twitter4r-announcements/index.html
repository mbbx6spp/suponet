Previously you would have only been able to get the last 20 sent/received direct messages from the Twitter4R bindings doing something like the following: %pre %code :preserve messages = client.messages(:sent) Now you can request specific pages like so: %pre %code :preserve messages = client.messages(:sent, :page => 2) The same is true for the %tt :received case.
