Which Default Ports?

This site provides information about default ports for services and applications. There are a few types of ranges of ports.

Reserved/well known port range (0-1023)

The ports in the range 0 to 1023 are used by sytem processes providing widely used network services. In the UNIX world a process must execute with "root" privileges to bind a network socket to a port in this range.

Registered port range (1024-49151)

This port range has default ports assigned to services and applications by the IANA. On UNIX and most other operating systems any user may bind a network socket to a port in this range.

Private and ephemeral port range (49152-65535)

This range does not have default ports registered with IANA as it is reserved for dynamically allocated or ephemeral ports automatically assigned by the operating system, system services or applications.

Used over TCP, UDP and sometimes SCTP on dynamically allocated for client communication or to free up a well-known service listening port and establish longer-living connection to a client.
